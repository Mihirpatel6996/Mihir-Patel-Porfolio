Corona Virus Analysis using SQL

  • Tech Stack: Sql, Prostresql
  • Github URL: Project Link

Analyzed a comprehensive COVID-19 dataset using SQL to derive meaningful insights about the spread of the virus, its impact on public health, and recovery trends across different countries and regions.

Implemented data cleaning techniques to handle missing values, ensuring data integrity by using SQL queries to update null entries and prepare the dataset for analysis.

Conducted various analyses, including monthly trends of confirmed cases, deaths, and recoveries, as well as identifying countries with the highest confirmed cases and reported deaths, using aggregate functions and ranking methods.

The analysis provided critical insights into the pandemic's progression, highlighting the severity of COVID-19 in different regions, the effectiveness of healthcare responses, and trends over time, which can inform public health strategies and policies.
